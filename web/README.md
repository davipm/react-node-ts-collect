# Web

Web Application with Rest API

### Prerequisites

Node.js

### Installing Server

```
npm i
```

### Installing Client
/client

```
npm i
```

### start server and client

```
npm run dev
```

### start only client

```
npm run client
```

### start only backend

```
npm run backend
```

## Built With

* React.js
* Node.js
* TypeScript
